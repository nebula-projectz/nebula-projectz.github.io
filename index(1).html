Features

Everything you need

Built from scratch with GTK4 and Python. No Electron, no bloat — just speed.

🎨

Windows 11 Theme

Authentic Win11 Explorer experience on Linux — folder icons, sidebar, toolbar, and navigation all match the real thing.

⚡

Native Performance

Built with GTK4 and libadwaita. No web technology, no wrapper — pure native speed with hardware-accelerated rendering.

📂

Tabbed Browsing

Multiple tabs with tab management. Open new folders in tabs, drag between them, and keep your workflow organized.

☁️

Cloud Integration

OneDrive and Google Drive appear automatically in your sidebar through GNOME Online Accounts. No extra setup.

🔍

Smart Search

Search files instantly. Filter by type, sort by date or size, and navigate with breadcrumb paths.

🎯

Multiple Themes

Switch between Windows 11, macOS, Nova, and Minimal themes. Each one is a complete visual overhaul, not just colors.
