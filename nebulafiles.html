Features
Everything you need
Built from scratch with GTK4 and Python. No Electron, no bloat.

Multiple Themes

Windows 11, macOS, Nova, Minimal, Retro. Each is an independent visual overhaul.

Cloud Integration

OneDrive and Google Drive via rclone — browse cloud files like local ones.

Tabbed Browsing

Multiple tabs, drag files between them, stay organized.

Native Performance

GTK4 + libadwaita. Hardware-accelerated, no web technology overhead.

Smart Search

Filter by type, sort by date or size, breadcrumb navigation.

Keyboard Shortcuts

Fully customizable shortcuts for power users.
