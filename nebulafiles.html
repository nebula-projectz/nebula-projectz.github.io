Everything you need

Built from scratch with GTK4 and Python. No Electron, no bloat.

🎨

Multiple Themes

Switch between Windows 11, macOS, Nova, Minimal, and Retro themes. Each is a complete visual overhaul.

☁️

Cloud Integration

OneDrive and Google Drive appear in your sidebar. Browse and manage cloud files like local ones.

📂

Tabbed Browsing

Open multiple folders in tabs, drag between them, and keep your workflow organized.

⚡

Native Performance

Built with GTK4 and libadwaita. Pure native speed with hardware-accelerated rendering.

🔍

Smart Search

Search files instantly. Filter by type, sort by date or size, navigate with breadcrumb paths.

🪟

Windows 11 Theme

Authentic Win11 Explorer experience on Linux — icons, sidebar, toolbar all match the real thing.
