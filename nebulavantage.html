Features

Everything you need

Built from scratch with GTK4 and Python. No Electron, no bloat.

🎵
Audio & Video
Plays MP3, FLAC, OGG, WAV, M4A, MP4, MKV, AVI and more via GStreamer.
📋
Smart Playlists
Folder-based playlists stored locally. Portable — copy the folder to any machine.
🖼
Custom Icons
Set a PNG, JPG, or emoji as icon for any playlist or track. Stored locally.
✏️
Edit Metadata
Pencil edit button on playlists and tracks — rename, change icon, update info.
🌙
Dark & Light Mode
Instant toggle between dark and light themes. Preference saved automatically.
⚡
Native Performance
GTK4 + libadwaita. Hardware-accelerated, no web technology overhead.
